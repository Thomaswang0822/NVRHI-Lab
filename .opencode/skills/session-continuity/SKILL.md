# Session Continuity Skill

## Purpose
Quickly review all context files to get up-to-date when starting a new session.

## Usage
When you (the AI agent) need to refresh context about the project, read through all `.md` files in `.opencode/contexts/` directory.

## What This Provides
- Project design and philosophy
- Choices made (backends, C++ standard, etc.)
- Learning purpose and boundaries
- Rules and conventions
- Technical stack

## How to Use
Run the script: `.opencode/skills/session-continuity/review-context.sh`

Or manually read all files: `.opencode/contexts/*.md`
